# Principal Component Analysis (PCA) from Scratch

## Project Overview

This project implements **Principal Component Analysis (PCA)** from scratch using only NumPy. The goal is to reduce the dimensionality of a real-world African dataset while preserving as much information as possible.

PCA is achieved through linear algebra concepts such as covariance matrices, eigenvalues, and eigenvectors, without using machine learning libraries like sklearn.

---

## Objectives

* Implement PCA manually using NumPy only
* Compute covariance matrix from standardized data
* Perform eigen decomposition to find principal components
* Project data onto lower-dimensional space
* Dynamically select the number of components based on explained variance
* Analyze tradeoffs between dimensionality reduction and information loss
* Optimize performance for larger datasets
